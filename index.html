Simplebox - a product by d-xp.com

... just a simple box.

Usage

$.simplebox.show(content, {
	//settings
	'title'     : false,
	'closeOnEsc': true,
	'theme'     : 'default',
	'height'    : 'auto',
	'width'     : 'auto',
	'speed'     : 500,
	'easing'    : 'swing',
	'buttons'   : false,

	//events
	'beforeShow'  : function(){},
	'beforeClose' : function(){},
	'onClose'     : function(){}
});
where content could be a string, dom element or jQuery element.

Shortcuts

$.simplebox.alert("...")

$.simplebox.confirm("...", function(){ 
	//your callback code
});
